# Cluster Simulations

Implementations of various cluster growth and percolation models.

## Models

- **Cluster Find.py** - Cluster identification algorithms
- **DLA Cluster.py** - Diffusion-Limited Aggregation
- **DLA Cluster Mod.py** - Modified DLA implementation
- **Eden Cluster.py** - Eden growth model
- **Epidemic Cluster.py** - Epidemic spreading simulation
- **Percolation Cluster.py** - Percolation theory simulation

These models explore growth patterns, fractal structures, and phase transitions in physical systems.
