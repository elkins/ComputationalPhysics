# PHYS 613 - Advanced Computational Physics

This directory contains coursework from PHYS 613, organized into assignments and projects.

## Structure

- **assignments/** - Weekly assignments (A1-A11) covering various computational physics topics
- **projects/** - Four major course projects

## Topics Covered

### Assignments
- Root finding algorithms (A1-A2)
- Quantum mechanics (square wells, wave functions) (A2-A3)
- Numerical integration and curve fitting (A4-A5)
- Random walks and Monte Carlo methods (A6-A7)
- ODEs and dynamical systems (A8)
- FFT and frequency analysis (A10)
- PDEs (heat equation, wave equation) (A11)

### Projects
- **Project 1**: Molecular potentials and optimization
- **Project 2**: Genetic algorithms
- **Project 3**: Cluster analysis and fractal dimensions
- **Project 4**: Perturbed oscillators
