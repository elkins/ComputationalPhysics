# PHYS 510 - Computational Physics Course

This directory contains assignments and projects from PHYS 510.

## Files

- **Final1 FukushimaExp.py** - Fukushima radiation leak data analysis with exponential models
- **Final1 FukushimaExpLog.py** - Alternative exponential/logarithmic fit models
- **Final2 FFT.py** - Fast Fourier Transform analysis and frequency domain processing
- **Final3 Electrostatics.py** - Electrostatic potential calculations
- **FoucaultPendulumGUI.py** - GUI simulation of Foucault pendulum
