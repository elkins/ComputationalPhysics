# Examples

Demonstration programs and utility scripts.

## Files

- **Concentric Circles Example.py** - Geometric visualization
- **MandelbrotSet.py** - Mandelbrot set fractal generation
- **Random Sphere Example.py** - Random point distribution on spheres
- **RotationConverter.py** - Rotation matrix and angle conversions
- **Scipy Optimize Example.py** - Optimization using SciPy library

These examples demonstrate various computational techniques and visualization methods.
